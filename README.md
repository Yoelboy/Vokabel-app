# Vokabel-App

Dies ist eine Vokabel-App, die dir hilft, neue Vokabeln zu lernen.

## Verwendung

- Bei "Vokabel Hinzufügen" kannst du neue Vokabeln eintragen.
- Bei "Vokabel Test" kannst du dein Wissen testen.
